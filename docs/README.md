# Tiwani Documentation Workspace

Canonical home for shared project documentation in the Option 3 monorepo.

```text
docs/
  architecture/     System and data architecture notes
  guides/           Engineering guides for the mobile app
  operations/       Operational runbooks
  planning/         Product planning and phase documents
  policies/         Policy drafts (privacy, terms, marketplace, support)
  production/       Production evidence and hardening notes
  release/          Release and store-launch guides
  *.md              Release checklists, handoffs, runbooks, and QA evidence
```

The mobile app source lives in `apps/mobile/`, and the Firebase workspace
lives in `backend/firebase/`. Historical documents in this folder may mention
the pre-migration `frontend/` path; treat `apps/mobile/` as its current
equivalent.
